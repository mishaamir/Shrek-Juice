<h1> Shrek Juice</h1>

<h2>Description</h2>
<p>
  Shrek Juice is an online juice store website that showcases a variety of fresh and healthy juices through a clean and visually appealing interface. The project is designed to present products effectively with a simple and user-friendly layout.
</p>

<h2>Technologies Used</h2>
<ul>
  <li>HTML5</li>
  <li>CSS3</li>
  <li>JavaScript</li>
  <li>Bootstrap 5</li>
</ul>

<h2>Features</h2>
<ul>
  <li>Modern and attractive user interface</li>
  <li>Product showcase section for different juice varieties</li>
  <li>Easy navigation with a well-structured layout</li>
  <li>Interactive elements using JavaScript</li>
  <li>Bootstrap components for enhanced design and functionality</li>
  <li>Contact section for customer inquiries</li>
  <li>Clean and organized frontend structure</li>
</ul>

<h2>Purpose</h2>
<p>
  This project demonstrates frontend web development skills using HTML, CSS, JavaScript, and Bootstrap, focusing on creating an attractive and functional product showcase website.
</p>
